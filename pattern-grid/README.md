# Pattern Grid

A geometric pattern editor for creating visual patterns using triangles and arcs.

## Features

- **Drawing Modes**: Toggle between triangle and arc patterns
- **Click-based Drawing**: Click near any corner of a cell to draw in that direction
- **Adjustable Grid**: Change grid size from 8x8 to 128x128
- **Color Selection**: Choose any color for your patterns
- **Export**: Download as PNG or copy to clipboard
- **Mobile Support**: Works on touch devices
